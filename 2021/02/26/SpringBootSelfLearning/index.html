Spring Boot 入門自學資源統整

Author: Louis Chang Date: February 26, 2021  23:14:31 Category: 技術筆記

這篇文章會提供我在自學Spring Boot 過程中用到的資源以及一些心得分享，之後也會出一篇較短的Spring Boot 地雷及拆彈指南，希望可以幫助大家在自學Spring Boot 的路上少走一點歪路！

在開始之前，我想先說明一下，以下提供的純屬我個人在自學過程中曾經使用過的資源，絕非業配（也不會有人找我業配QQ），如果你還知道其他不錯的資源也歡迎提供給我！第一部分主要會是我對於Udemy 這個課程的個人心得，如果只是想看自學資源的話可以直接跳到後面～

課程心得分享

Spring Boot從入門到高級實戰課程

為什麼我會想學Spring Boot

我在網路上看過很多大神的文章，學東西常常只是因為興趣、好玩，沒有什麼特別的目的，然後莫名其妙就變超強了（好羨慕）。但很不幸的是，我只是個平凡人，主要是因為之後實習工作內容會用到，Java 又是我接觸比較久的語言，再加上對於框架又沒什麼概念（我好爛QQ），所以就在茫茫框架之海中選了Spring Boot！

為什麼我會選這個課程

現在市面上有非常多個線上課程的平台，資訊相關課程的話很多人會選擇Coursera、Udemy、Codecademy、Udemy等等，而我這次選擇的是Udemy，因為他很便宜！（沒錯，就是這麼膚淺的原因）平常官網上的價格區間大概在兩千到五千，但折扣的時候只需要三百多！（你沒看錯，是一折不是九折）這門課程原本定價是3690，而我當時只花了370就買到了這門課。除此之外，Udemy還有提供30天退款保證，如果看了不喜歡還有後悔的機會（如果你很猛的話當然也能在30天內速速看完然後退款）。所以如果你之後也想買Udemy的課程的話，請務必選在特價的時候，我印象中滿常有折扣的，這部分就要麻煩你自己去查一下囉，然後FB有些社團會公布折扣或是限免的訊息，可以加入看看。

課程心得

對於這門課我個人推薦程度是4顆星（滿分5顆）！

這是由一個中國的老師開的，課程語言以簡體中文為主，老師講解的算滿清楚、容易理解，也會一步一步的帶著打code，滿適合Spring Boot 的初學者的，但還是建議有一點Java的基礎再學比較好，因為課程裡面比較沒有提到一些Java的基礎觀念。此外，課程的架構是以小專案作為主軸，帶著學生使用各種會用到的東西，個人滿喜歡這種上課模式。

講完優點就來說說一些小缺點吧，首先是老師的英文發音真的是不太好，有時候已經不是口音的問題了，是直接把單字整個念錯，所以如果你對於英文發音很敏感的話不建議買這門課。再來是code版本問題，因為這門課已經上架好一陣子了，有些課程裡面的函式用法跟現在不太一樣，編譯不給過的話就需要自己再查一下囉～
